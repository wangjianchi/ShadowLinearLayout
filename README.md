# ShadowLinearLayout
一个带阴影的类似CardView的控件
![iamge](https://github.com/wangjianchi/ShadowLinearLayout/blob/master/shortcut/shortcut01.png)
###属性
    shadowColor; //阴影颜色
    bgColor; //背景颜色
    shadowRadius; //阴影半径
    offset_x; //阴影x轴偏移量
    offset_y; //阴影y轴偏移量
    radius_x; //圆角半径x
    radius_y; //圆角半径y
